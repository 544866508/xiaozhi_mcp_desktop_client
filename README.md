# 介绍
基于Windows的可视化MCP管理桌面应用

<p>本工具为小智MCP标准服务WebSocket桥接客户端，提供windows可视化GUI管理面板，面向小白支持一键安装，支持多MCP服务统一启停、状态可视化展示，MCP工具可视化编辑、启停状态自动同步。大幅简化小智MCP服务对接流程，具体优势如下 👇</p>


1. **一键安装，隔离Python依赖，杜绝全局包冲突**
全自动安装python与虚拟环境依赖，不污染本机自带Python环境，即使有多套基于python的AI、DIY项目也互不干扰。
2. **纯Windows完整支持**
官方仅提供Linux部署教程，大量无软件开发经验的玩家无从下手，本工具完整适配Windows系统，开箱即用。
3. **实时MCP进程监控，杜绝僵尸进程**
自带运行状态可视化面板，区分在线/离线进程，显示PID，一眼识别假启动、僵尸进程。
4. **全图界面操作**
可视化增删改查、批量编辑配置，小白也能快速上手。

<p>本工具无任何收费功能，如果帮你节省了开发时间、解决了问题，欢迎 去B站三连支持一下 或者去 GitHub 点个 Star</b>，你动动手指就是对我最大的鼓励。</p>


# 已实现功能
1. 客户端一键安装启动
2. 图形界面管理MCP源，支持单独修改或批量修改MCP源
3. MCP进程监控
4. 接入多个MCP
5. 图形界面导入WSS地址
